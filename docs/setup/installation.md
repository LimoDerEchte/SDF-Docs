# Installation

### 1. Install UE4SS
If not already done, install UE4SS from [Nexus Mods](https://www.nexusmods.com/subnautica2/mods/36?tab=files).

### 2. Download the Mod
The mod can be downloaded from the [Nexus Mods downloads page](https://www.nexusmods.com/subnautica2/mods/383?tab=files).</br>
From the Nexus page you should download the newest version of SDF available.

### 3. Extract the downloaded `SDF.zip` file

### 4. Copy the SDF folder into the UE4SS `Mods` directory
The `Mods` directory can be found in the `/Subnautica2/Subnautica2/Binaries/Win64/ue4ss/` folder.</br>
When done correctly, you should now be able to find an `enabled.txt` file at `/Subnautica2/Subnautica2/Binaries/Win64/ue4ss/Mods/SDF/enabled.txt`.

## Happy modding!
